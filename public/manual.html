01

단톡방에서 초대코드 받기

카카오 오픈채팅 단톡방 공지사항에서 초대코드를 확인할 수 있어요.

💬 카카오 단톡방 입장하기

베이직 가입 방법

1
앱 접속 → 가입 탭초대코드 + 닉네임 + 비밀번호 입력
2
주제 입력 → AI 글 생성원하는 주제 입력 후 생성 버튼을 누르면 바로 글이 나와요
3
복사해서 Threads에 직접 올리기베이직은 글 생성을 하루 200번 무제한으로 쓸 수 있어요
PART 02 — 프로 토큰 발급
02

Meta 개발자 계정 만들기

1
developers.facebook.com 접속우측 상단 로그인 → 페이스북 계정으로 로그인
2
내 앱 → 앱 만들기앱 유형: 기타 선택 → 다음
3
앱 이름 입력 → 앱 만들기이름은 아무거나 입력해도 돼요
03

Threads API 설정

1
앱 대시보드 → 제품 추가Threads API 찾아서 설정 클릭
2
앱 시크릿 복사설정 → 기본 설정 → 앱 시크릿 코드 복사 후 메모장 저장
3
테스터 추가앱 역할 → 역할 → Threads 테스터에 내 계정 추가
04

테스터 초대 수락

Threads 앱에서 직접 수락해야 해요!

1
Threads 앱 열기설정 → 계정 → Meta 테스터 초대 탭
2
초대 수락 클릭수락 안 하면 토큰 발급이 안 돼요!
05

단기 토큰 발급

아래 URL의 앱ID를 교체해서 브라우저에 입력해요.

https://threads.net/oauth/authorize?client_id=앱ID&redirect_uri=...&response_type=code

승인 후 주소창에서 code= 뒤 값을 복사해요

06

단기 → 장기 토큰 변환

⚠️ 주의사항
· 띄어쓰기 절대 없어야 함
· 한글 텍스트 남아있으면 안 됨
· 단기 토큰은 발급 후 1시간 이내 사용
https://graph.threads.net/access_token?...client_secret=앱시크릿&access_token=단기토큰
1
메모장에서 교체 후 브라우저 입력앱시크릿, 단기토큰 값으로 교체
2
JSON 응답에서 access_token 복사따옴표 안쪽 값만 → 60일짜리 장기 토큰이에요
07

앱에 토큰 등록

1
사이드바 → 계정 관리 (프로 전용)왼쪽 메뉴에서 계정 관리 클릭
2
계정 이름 입력메인 계정, 쿠팡 계정 등 구분하기 쉽게
3
장기 토큰 붙여넣기 → 계정 추가완료! 이제 자동 발행을 쓸 수 있어요 🎉

PART 04 — 자주 묻는 질문

베이직은 언제까지 쓸 수 있나요?

기간 제한 없이 영구 무료로 사용할 수 있어요. 만료 걱정 없이 글 생성+복사를 계속 쓸 수 있어요.

토큰은 얼마나 자주 발급해야 해요?

60일마다 재발급이 필요해요. 만료 전에 미리 새 토큰으로 교체해주세요.

이모지나 해시태그가 들어가요

재생성 버튼을 누르면 달라진 글이 나와요. 서버에서 자동 제거 처리도 돼 있어요.

프로 기간이 끝나면?

자동으로 베이직으로 내려가요. 글 생성+복사는 계속 쓸 수 있고, 발행 기능만 비활성화돼요. 카카오로 문의하면 바로 재활성화돼요.

발행이 안 돼요

1. 토큰 만료 여부 확인 (60일)
2. 계정 선택 여부 확인
3. 그래도 안 되면 카카오로 문의해줘요
