🔑 장기 액세스 토큰 발급 방법

📌 순서대로 따라하세요
1단계 — Meta 개발자 콘솔 설정
① developers.facebook.com 접속 → 앱 생성
② 앱 유형: 소비자 선택
③ 왼쪽 메뉴 → Threads API 액세스 → 설정 탭
④ 사용자 토큰 생성기 섹션에서 Threads 테스터 추가
2단계 — 단기 토큰 발급
① 사용자 토큰 생성기에서 액세스 토큰 생성하기 클릭
② 단기 토큰 복사 (1시간짜리)
3단계 — 장기 토큰으로 교환 (60일짜리)
아래 URL을 브라우저 주소창에 붙여넣기 (앱시크릿과 단기토큰 교체)
https://graph.threads.net/access_token?grant_type=th_exchange_token&client_secret=앱시크릿&access_token=단기토큰
✅ 응답에서 access_token 값 복사 → 아래 토큰란에 붙여넣기
