👤 계정 관리

장기 액세스 토큰 발급 방법

1단계 — developers.facebook.com 앱 생성 → Threads API → 사용자 토큰 생성기
2단계 — 액세스 토큰 생성하기 클릭 → 단기 토큰 복사
3단계 — 아래 URL에 앱시크릿, 단기토큰 교체 후 브라우저 주소창에 입력
https://graph.threads.net/access_token?grant_type=th_exchange_token&client_secret=앱시크릿&access_token=단기토큰
응답에서 access_token 값 복사 → 아래 토큰란에 붙여넣기
